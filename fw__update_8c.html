Detailed Description

Firmware Update for Synapticon SOMANET (modified from Example code for Simple Open EtherCAT master)

Usage: fw_update eth0 OPTION id filepath OPTION: -s specify serial number for the ethercat slave -seq specify slave number in EtherCAT order 1..n -all use all slaves connected to the system -scan scan the slave/slaves connected and display their serial number

specify the ID number after options for these options -s & -seq followed by /path/to/your/file
